# Banking Simulator (C)

A console-based Banking Simulator in C.  
Users can create accounts, deposit, withdraw, and check balances.  
All data is stored in `account.dat` with a clean, colored console interface.

## Features
- Create accounts  
- Deposit money  
- Withdraw money  
- Check account balance  
- File-based storage  

## Run
Compile and run on Windows. The program auto-creates `account.dat`.
